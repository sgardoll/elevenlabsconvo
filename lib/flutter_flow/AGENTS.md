## Model Pattern

Widget models extend `FlutterFlowModel`:

```dart
class MyWidgetModel extends FlutterFlowModel<MyWidget> {
  @override
  void initState(BuildContext context) {}
  
  @override
  void dispose() {}
}
```

Use `wrapWithModel()` in `build()`:

```dart
wrapWithModel(
  model: _model.myComponentModel,
  updateCallback: () => setState(() {}),
  child: MyComponent(),
)
```
